# 수직 슬라이스 검증 베이스라인

## 대상

- 구현 범위: `Legacy Java EE` 최소 수직 슬라이스
- 프로파일: `legacy-java-ee`
- 샘플 입력: `samples/legacy-java-ee-minimal`

## 현재 통과 기준

- `build.xml` 기반 모듈/배포 단위 식별
- `web.xml` 기반 웹 진입점 식별
- Spring XML 기반 bean 선언 식별
- Java 소스 기반 `@RequestMapping` 및 controller/service/dao 후보 식별
- MyBatis/iBATIS 기반 mapper/sql statement 식별
- JSP/SiteMesh 기반 view/layout 식별
- `Start Here`의 `Primary Entry Flows` 노출 확인
- `Entry Flow` 탭의 `URL -> Controller/Action -> View -> Layout` 후보 확인
- `가로형/카드형` 보기 전환 동작 확인
- lazy render와 panel cache로 탭 전환 시 재렌더 범위가 제한되는지 확인
- 결과 병합 후 그래프 JSON 생성
