# 부트스트랩 계획

## 현재 마일스톤

첫 마일스톤은 아래 전체 수직 슬라이스가 끝까지 도는 상태다.

- `profile detection`
- `4개 핵심 어댑터 실행`
- `normalized graph JSON 생성`
- `merge`
- `진입 흐름 중심 리포트 출력`

## 구현 순서

1. 공통 코어 타입
2. 어댑터 인터페이스와 실행기
3. Legacy Java EE 프로파일
4. `ant-build-xml`, `web-xml`, `spring-xml`, `java-source-basic`
5. 병합 및 로컬 저장
6. fixture 테스트
7. 진입 흐름 UI 정리

## 초기 제외

- 웹 UI
- 협업 워크플로우
- DB 저장소
- 증분 분석
- 다중 프로파일 동시 고도화
- 대시보드형 협업 워크플로우
